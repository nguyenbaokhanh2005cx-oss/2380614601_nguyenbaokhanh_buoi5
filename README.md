## Chạy project

1. Cài .NET 9 SDK và SQL Server (hoặc LocalDB).
2. Sửa connection string trong `Buoi5/appsettings.json` cho đúng máy bạn.
3. Trong thư mục gốc repo:

```bash
cd Buoi5
dotnet run
```

4. Mở trình duyệt: `http://localhost:5153`
